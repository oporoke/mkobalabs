## Step 32 — Create GitHub Actions workflow

On your **local computer**, clone the repository:



```bash

```

Put this inside:

```yaml
name: Deploy Mkoba API

on:
  push:
    branches:
      - master

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:
      - name: Deploy to VPS
        uses: appleboy/ssh-action@v1.0.3
        with:
          host: 173.249.48.40
          username: root
          key: ${{ secrets.VPS_SSH_KEY }}
          script: |
            cd /var/www/myapp
            git pull origin master
            npm install --production
            pm2 restart mkoba-app
```
